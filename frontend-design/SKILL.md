此技能用于指导打造独具特色、可投入生产的前端界面，规避俗套的“人工智能劣质审美”。编写可实际运行的代码，对美学细节与创意设计做到极致考究。

用户会提供前端开发需求：需构建的组件、页面、应用程序或界面。需求中可包含用途、目标用户或技术限制等相关背景信息。

## 设计思路

编码前，先理清需求背景，并确立鲜明大胆的设计风格方向：

- **用途**：该界面解决什么问题？用户群体是谁？
- **基调**：选择极致风格：极简冷峻风、极繁杂乱风、复古未来风、自然原生风、奢华精致风、趣味童趣风、杂志排版风、粗野原始风、装饰艺术几何风、柔和马卡龙风、工业实用风等。风格种类丰富，可从中汲取灵感，但最终设计需贴合选定的风格方向。
- **限制**：技术要求（框架、性能、无障碍适配）。
- **差异化**：是什么让设计令人过目不忘？有哪个核心亮点能让人印象深刻？

**关键要求**：选定清晰的设计理念方向，并精准落地执行。大胆的极繁设计与克制的极简设计均可，核心在于设计的目的性，而非视觉复杂度。

随后编写可运行代码（HTML/CSS/JS、React、Vue 等），代码需满足：

- 生产可用、功能完整
- 视觉惊艳、令人难忘
- 风格统一、设计理念清晰
- 细节打磨精益求精

## 前端美学准则

重点关注以下方面：

- **排版**：选用美观、独特、有辨识度的字体。选择能提升界面质感的特色字体，采用出人意料、富有个性的字体搭配。将一款特色标题字体与一款精致正文字体组合使用。
- **色彩与主题**：确立统一的美学风格，通过 CSS 变量保证视觉一致性。主色调搭配鲜明点缀色，远优于保守、分布均匀的配色方案。
- **动效**：运用动画实现视觉效果与微交互。HTML 优先使用纯 CSS 动效，React 可搭配 Motion 动画库。聚焦高冲击力场景：一个编排精良、带错落渐显效果的页面加载动画，比零散的微交互更具愉悦感。使用滚动触发、悬停状态等惊喜动效。
- **空间布局**：采用非常规布局、不对称设计、元素叠加、斜向动线、突破网格的设计元素。合理运用大量留白，或控制元素密集度。
- **背景与视觉细节**：营造氛围与层次感，而非默认使用纯色背景。添加契合整体风格的场景化效果与纹理，运用渐变网格、噪点纹理、几何图案、多层透明叠加、强烈阴影、装饰边框、自定义光标、颗粒质感叠加等创意形式。

**严禁使用**：通用人工智能生成式审美，例如滥用俗套配色方案（尤其是白底紫色渐变），千篇一律的布局与组件模板，以及缺乏场景特色的流水线式设计。

进行创意解读，做出贴合场景、出人意料的设计选择。杜绝设计同质化，在明暗主题、字体选用、风格方向上灵活变化。

**重要说明**：代码实现复杂度需匹配设计视觉效果。极繁风格设计需搭配复杂代码，实现丰富动画与特效；极简或精致风格设计则需克制、精准，着重把控间距、排版与细腻细节。优雅源于对设计理念的完美落地。

谨记：你具备出色的创意创作能力。大胆突破，充分展现跳出固有思维、全力践行独特设计理念所能打造的极致成果。
